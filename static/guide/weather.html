Weather

Χρησιμοποιεί open-meteo για τρέχουσα θερμοκρασία και πρόγνωση.

Χρήση

  1. Δώσε γεωγραφικό πλάτος/μήκος στα blocks (ή χρησιμοποίησε τα default της θέσης σου).
  2. Κάλεσε current temperature ή forecast for [hours].
  3. Οι τιμές έρχονται σε °C.

Συμβουλές

  • Οι κλήσεις είναι χωρίς κλειδί API (open-meteo).
  • Για διαφορετικές μονάδες, μετατρέψτε στο project (π.χ. C→F).
