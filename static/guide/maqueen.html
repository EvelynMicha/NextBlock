Maqueen Plus V2

BLE UART extension για έλεγχο κίνησης, LEDs, line sensors, ultrasonic και IR.

Firmware

Φόρτωσε στο micro:bit (πάνω στο Maqueen) το αρχείο maqueen-plus-v2.hex. Θα το βρεις μόλις προστεθεί στη σελίδα firmware:

Σελίδα firmware

Σύντομες οδηγίες flash: σύνδεσε USB → σύρε το .hex στο drive MICROBIT → περίμενε να σταματήσει να αναβοσβήνει το LED.

Σύνδεση

  1. Άνοιξε το NextBlock και πρόσθεσε το extension Maqueen Plus.
  2. Πάτησε connect (Bluetooth) και επίλεξε το micro:bit.
  3. Όταν το block maqueen connected? γίνει true, είσαι έτοιμος.

Blocks (MVP)

  • Κίνηση: set [motor L/R/both] [forward/backward] speed (0-255), stop [motor]
  • LEDs: front LED [left/right/both] [on/off]
  • Line sensors: line sensor [L1/L2/M/R1/R2] state, line sensor [L1/L2/M/R1/R2] analog
  • Ultrasonic: distance (cm)
  • Info: firmware version, last IR code

Πρωτόκολλο (BLE UART)

  • MOTOR:<L|R|B>:<F|B>:<0-255>
  • STOP:<L|R|B>
  • LED:<L|R|B>:<0|1>
  • LINE?:<L1|L2|M|R1|R2> → LINE:<id>:<0|1>
  • LINEDATA?:<...> → LINEDATA:<id>:<value>
  • ULTRA? → ULTRA:<mm>
  • VERSION? → VER:<string>
  • IR:<code> (event σε αλλαγή)

Default pins

  • Ultrasonic: P13 (trig), P14 (echo)
  • Line sensors: L1/L2/M/R1/R2 (όπως στο firmware)
  • RGB strip: απενεργοποιημένα blocks στη συγκεκριμένη έκδοση

Συμβουλές

  • Αν δεν παίρνεις απάντηση στο PING?, ξανακάνε pairing.
  • Για IR remote, το last IR code ενημερώνεται μόνο σε αλλαγή κωδικού.
