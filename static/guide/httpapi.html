HTTP / API

Απλά blocks για fetch JSON/text από δημόσια APIs.

Παράδειγμα (adviceslip)

  1. fetch JSON from https://api.adviceslip.com/advice
  2. set [h v] to (last json handle)
  3. set [adviceText v] to (property [advice] of (property [slip] of (h)))
  4. say (adviceText) for 5 seconds

Συμβουλές

  • Χρησιμοποίησε HTTPS URLs.
  • Οι κλήσεις είναι CORS-limited: χρειάζεται public API με CORS open.
