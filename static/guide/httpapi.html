HTTP / API

Κάνει αιτήματα GET σε δημόσια APIs και επιστρέφει JSON ή κείμενο.

Βήματα

  1. Βάλε URL στο block fetch url [ ].
  2. Χρησιμοποίησε with header key:value αν χρειάζονται headers.
  3. Διάβασε την απάντηση με response body (κείμενο) ή response json [path].
  4. Έλεγξε status code για επιτυχία (200).

Συμβουλές

  • Πρέπει το endpoint να επιτρέπει CORS.
  • Για μεγάλες απαντήσεις, φιλτράρισε με JSON path (π.χ. weather[0].temperature).
