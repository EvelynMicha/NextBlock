EasyPlug

Extension για το EasyPlug shield: WebSerial (USB) + Radio (χωρίς Bluetooth). Έλεγχος pins, LCD1602 (I2C), TM1637 και ανταλλαγή μηνυμάτων μέσω radio.

Firmware

  • microbit-nextblock_easyplug_radio.hex
  1. Σύνδεσε micro:bit με USB.
  2. Κατέβασε το .hex και αντέγραψέ το στον δίσκο MICROBIT.
  3. Μετά το flash άνοιξε το NextBlock, κάνε Σύνδεση μέσω WebSerial.

Ραδιο-εντολές

  • set radio group [n] – ορίζει group (0–255).
  • radio send string [text] / radio send number [n] – στέλνει μήνυμα.
  • radio has message? – boolean, αν υπάρχει ουρά.
  • radio receive – επιστρέφει το παλιότερο ληφθέν μήνυμα και το αφαιρεί.

Στο firmware προστέθηκαν εντολές RADIOSET:<group>, RADIOSEND:<payload>, και τα εισερχόμενα στέλνονται πίσω στη σειριακή ως RADIO:<payload>.

Pins / LCD / TM1637

  • Ψηφιακά: DW:P0:1, DR:P0?
  • Αναλογικά: AW:P1:512, AR:P1?
  • LCD1602: LCDINIT:39, LCDSTR:39:0:0:Hello, LCDNUM:39:0:1:123, LCDCLR, LCDON, LCDOFF
  • TM1637 (CLK=P2, DIO=P1 default): TMINIT:P2:P1:7:4, TMNUM:1234, TMHEX:0x1A2B, TMBIT:pos:digit, TMDP:pos:1, TMCLEAR, TMON, TMOFF
  • Αισθητήρας θερμοκρασίας: SENSE:TEMP? → SENSE:TEMP:<val>

Συμβουλές

  • Απενεργοποιήσαμε Bluetooth στο firmware για σταθερότητα. Χρησιμοποίησε USB + WebSerial.
  • Το radio group default είναι 1. Αν υπάρχουν πολλά micro:bit στον ίδιο χώρο, όρισε διαφορετικά group.
  • Για TM1637 / LCD χρειάζονται τα αντίστοιχα MakeCode packages (I2C_LCD1602, TM1637) όταν κάνεις build custom firmware.
