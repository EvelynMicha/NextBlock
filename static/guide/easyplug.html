EasyPlug

Extension για το EasyPlug shield: WebSerial (USB) ή BLE UART. Pins, LCD (I2C), TM1637.

Firmware

Κατέβασε το firmware και φλαςάρισε το micro:bit που κουμπώνει στο EasyPlug:

Κατέβασε microbit-nextblock_easyplug_super.hex

Οδηγίες flash: σύνδεση USB → drag & drop στο MICROBIT → περίμενε να ολοκληρωθεί.

Σύνδεση

  1. Άνοιξε το NextBlock και πρόσθεσε το extension EasyPlug Super.
  2. Πάτησε connect (USB) (WebSerial) ή connect (Bluetooth) για BLE.
  3. Το block easyplug connected? πρέπει να γίνει true.

Βασικά blocks

  • digital write [PIN] to [0/1], digital read [PIN]
  • analog write [PIN] to (0-1023), analog read [PIN]
  • LCD: init διεύθυνση, show string/number, clear, on/off
  • TM1637: init (CLK=P2, DIO=P1, brt=7, digits=4), show number/hex/digit/DP, clear/on/off
  • Temperature reporter: temperature (°C)
  • LED helper: set LED [on/off] on pin [PIN]

Πρωτόκολλο (USB/BLE)

  • DW:<pin>:<0|1>, DR:<pin>? → DR:<pin>:<0|1>
  • AW:<pin>:<0-1023>, AR:<pin>? → AR:<pin>:<value>
  • SENSE:TEMP? → SENSE:TEMP:<val>
  • LCD: LCDINIT, LCDSTR, LCDNUM, LCDCLR, LCDON/OFF
  • TM1637: TMINIT, TMNUM, TMHEX, TMBIT, TMDP, TMCLEAR, TMON/TMOFF

Πινάκια (συμβουλές)

  • Digital: P0, P1, P2, P3, P4, P5, P6, P7, P8, P10, P11, P12.
  • Analog: P0, P1, P2, P10.
  • LCD: διεύθυνση συνήθως 39 (0x27).
  • TM1637: default CLK=P2, DIO=P1.
