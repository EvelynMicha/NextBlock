Teachable Machine Image

Φόρτωσε TM image model (.json) και κάνε ταξινόμηση από την κάμερα.

Βήματα

  1. Πάτησε load model from URL και δώσε το link του TM model.
  2. Ενεργοποίησε κάμερα (άδεια) και χρησιμοποίησε τα blocks classify image ή τα hat when class changes.
  3. Διάβασε label και confidence reporters.

Συμβουλές

  • Το μοντέλο πρέπει να είναι δημοσιευμένο (TM → Export → Upload). Χρησιμοποίησε το URL model.json.
  • Καλό φωτισμό και σταθερό υπόβαθρο για καλύτερα αποτελέσματα.
