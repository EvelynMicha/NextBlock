Teachable Machine Image

Φορτώνει TM image model (Keras/.json) και κάνει classify webcam ή εικόνες της σκηνής.

Βήματα

  1. Άνοιξε το extension AI Image (TM).
  2. Χρησιμοποίησε το block load model from [upload/URL] και διάλεξε το model.json από TM export.
  3. Τρέξε classify [camera] και διάβασε top label, top confidence.

Χρήσιμες συμβουλές

  • Προτίμησε μικρό input size (224px) για ταχύτερη εκτέλεση.
  • Σε αδύναμα μηχανήματα, βάλε sampling κάθε 200-300ms.
